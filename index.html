OPERATING SYSTEM

Operating System (OS) एक software होता है जो ki computer तथा यूज़र के mid interface ki तरह work करता है। इसे system software कहते है.operating system Instruction का समूह होता है जो ki storage device ma रहता है। तथा यह programs का समूह होता है जो ki computer के resources तथा operations को manage करता है.OS computer ma लोड होने वाला यह पहला program होता है। इसे program of programs भी कहते है.OS computer के सभी operations को manage करता है।

COMPUTER ARCHITECTURE

Computer के विभिन्न Components एवं उनके मध्य सम्बन्ध को कम्प्यूटर की संरचना (Computer Architecture) कहते हैं। लगभग सभी Computers की संरचना एक ही तरह की होती है।

COMPUTER NETWORK

Network से हमारा तात्पर्य कंप्यूटर्स व इसके साथ जुड़े हुए पेरीफेरल डिवाइसेज को किसी संचार माध्यम (Transmission media) से इस प्रकार जोड़ देना है कि उनमें से प्रत्येक User अर्थात computer के मध्य स्वतंत्र रूप से files, सूचनाओं, संदेशों आदि का आदान-प्रदान हो सके, और साथ ही एक दूसरे के संसाधनों और सुविधाओं को भी share (साझा) कर सकें। एक नेटवर्क में जोड़े जाने वाले कंप्यूटर हमारे पास भी स्थित हो सकते हैं और दूर-दूर जगहों पर भी बिखरे हो सकते हैं।जब दो या दो से अधिक Computers व अन्य पेरीफेरल डिवाइसेज को आपस में किसी कम्युनिकेशन मीडिया (संचार माध्यम) के माध्यम से जोड़ दिया जाता है तो उसे Computer Network कहते हैं।
